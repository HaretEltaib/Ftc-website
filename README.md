🚀 Next.js Project Setup Guide

This project was built using Next.js
 and initialized with the official create-next-app tool to provide a modern, scalable, and high-performance web application structure.

📦 Installation & Running the Project

Before starting, make sure you have Node.js
 installed on your system.

Start the Development Server

Run one of the following commands inside the project directory:

npm run dev

or

yarn dev

or

pnpm dev

or

bun dev
🌐 Open the Application

After running the server, open your browser and visit:

http://localhost:3000

The page will automatically reload whenever you edit the project files.

🛠️ Project Structure

The main page can be edited from:

app/page.tsx

This project uses the App Router architecture provided by Next.js for better scalability and performance.
